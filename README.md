# Umbrella Budget

Web-based quote generator for **Umbrella Custom** — a custom t-shirt company. Fill in client details, add products with quantities per size, and generate a print-ready PDF quote.

## Goal

Replace manual quoting with a fast, standardized flow: the salesperson fills out a form, the system automatically calculates the price tier based on total volume, and opens a print/PDF-ready window with the company's branding.

## Features

- Two-tab flow: **Quote** (quick PDF generation) and **Register Order** (sends data to Notion)
- PF or PJ client registration (CPF/CNPJ) with billing and delivery addresses
- Automatic address fill via [ViaCEP](https://viacep.com.br)
- Product selection by category, color, and size
- Automatic price tier calculation based on total volume (25–49 / 50–99 / 100–199 / 200–499 / 500+)
- Automatic surcharge for special sizes
- Real-time quantity and subtotal summary per item
- Sequential quote numbering per year (e.g. `UC-2026-0001`), persisted in `localStorage`
- PDF generation via the browser's print dialog (no external library)
- Notion integration — sends full order data (client info, items, PDF attachment)
- PDF upload support for attaching the generated quote to the order registration

## Tech stack

- **TypeScript** (strict, ES2020) — frontend and server
- **esbuild** — frontend bundler
- **Express** — HTTP server + Notion API proxy
- **Notion API** — order database
- **ViaCEP** — address auto-fill by postal code
- **ESLint 8 + Prettier** — linting and formatting

## Local setup

**Requirements:** Node.js 22.9.0 (run `nvm use` at the project root)

```bash
# Install dependencies
npm install

# Development mode (hot reload)
npm run dev

# Production build
npm run build

# Start server (after build)
npm start
```

The app runs at `http://localhost:3000`.

## Environment variables

Create a `.env` file at the project root (never commit this file):

```env
NOTION_TOKEN=secret_...
NOTION_DATABASE_ID=...
```

These are required for the **Register Order** tab. The Quote tab works without them.

## Deploying to Render

The [render.yaml](render.yaml) file configures automatic deployment:

- **Build:** `npm install && npm run build`
- **Start:** `npm start`
- **Env:** `NODE_ENV=production`

Set `NOTION_TOKEN` and `NOTION_DATABASE_ID` as environment variables in the Render dashboard. Connect the repository and every push triggers a new deploy automatically.

## Price tiers

Prices vary based on the total number of pieces across all items:

| Tier   | Quantity        |
|--------|-----------------|
| 25–49  | Minimum order   |
| 50–99  | —               |
| 100–199 | —              |
| 200–499 | —              |
| 500+   | Best price      |

Orders below 25 pieces do not generate a quote.
